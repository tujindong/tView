# Icon 图标

提供了一套常用的图标集合。

# 使用方法

直接通过设置类名为 tu-icon-iconName 来使用即可。例如：

<template>
	<div class="icon-block-demo">
		<i class="tu-icon-edit"></i>
		<i class="tu-icon-share"></i>
		<i class="tu-icon-delete"></i>
	</div>
</template>

<style>
	.icon-block-demo [class*="tu-icon-"] {
		margin: 0 20px;
		font-size: 1.2em;
		vertical-align: middle;
		color: #9baacf;
	}
</style>
